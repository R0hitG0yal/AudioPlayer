# Music Player

## Setup

1. Clone the repository
   ```bash
   git clone <repository-url>
   cd music-player

2. npm install

3. npm run dev

4. You will find the application running in the following URL:
http://localhost:3000

The application is deployed on Vercel. You can access it on https://audio-player-mu-eight.vercel.app/.

This setup should give you the basic functionality required. If you have more specific questions or need further customization, feel free to ask!
